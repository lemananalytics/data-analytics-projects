# İşçi Maaşlarının OLS Reqressiya və Statistik Analizi

Bu layihədə işçilərin maaş göstəriciləri, təcrübə illəri, performans balları və təlim saatları arasındakı əlaqələr statistik testlər və OLS Reqressiya modeli ilə analiz edilmişdir.

## Görülən İşlər:
Statistik Testlər: Shapiro-Wilk testi ilə datanın normallığı, Müstəqil T-Test ilə cinsiyyətlər arası maaş fərqi yoxlanıldı.
Korrelyasiya Analizi: Maaş ilə təcrübə ili (0.99) arasında mükəmmələ yaxın düz mütənasib əlaqə aşkarlandı.
OLS Reqressiya Modeli: Maaş dəyişkənliyini 99.2% dəqiqliklə izah edən xətti model quruldu.
Multikollinearlıq Təhlili: Dəyişənlərin öz aralarındakı güclü əlaqənin modelə təsiri sənədləşdirildi.
